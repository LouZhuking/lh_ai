## shrink 收缩计算
- 溢出空间 300px 让子元素来 shrink 分摊
- 分摊的比列？
  flex-shrink: 2 : 1  500:400

  500-300*500 需要分摊的比例
  400-300*400 需要分摊的比例
- 分摊的权重 
  总的权重？ 
  500*2 + 400*1 = 1400
  1000/1400 = 0.7143  71.43%
  400/1400 = 0.2857  28.57%

  300 * 0.7143 = 214.29
  300 * 0.2857 = 85.71

  left 500 - 214.29 = 285.71
  right 400 - 85.71 = 314.29
